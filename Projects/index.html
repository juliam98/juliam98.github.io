Hello

Here are some of my coding projects:

Photo to excel converter 

This script allows you to transfer your photo to excel by pixelating it into 64x64 rersolution and extracting the colour value of each pixel. The cells in excel are scaled into aquares and coloured with the colour of the corresponding pixel.
PDF converter GUI 

This is one of those projects you do out of spite. I refuse to pay for Acrobat premium to be able to rotate or merge PDFs. This simple GUI rotates individual pages and merges PDFs. This was my first go at object oriented prrogramming.
TFL API

This simple python script fetches departure times of an underground line at your stop from the TFL website. This is not yet on my github because (unless you also work 8 minutes away frorm Denmark Hill station) it's only useful for me. I will tweak it and upload soon.
Granny squares colours

I wrote this code when I wanted to make sure that the granny suqares I make for my blanket are as colourful as possible. This code takes the input of RGB colour values and generates all possible combinations of those colours in a granny square that consists of a specified number of smaller squares.
Internet speed report

A quarto report testing the differences in internet speed between devices. Inspired by Hyperopric support suggesting that the failed attempts to troubleshoot low internet speed are due to the device I used to take the measurements. Written in R, this report puts this claim to the test with ANOVA.
